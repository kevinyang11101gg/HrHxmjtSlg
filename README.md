# 项目简介

Python错题管理系统是一款基于Flask、MySQL、HTML、CSS和JavaScript开发的系统，旨在帮助用户管理和复习编程学习中的错题。系统包含详细的开发文档，方便用户进行配置和使用。

## 核心功能

- 用户登录与注册
- 错题信息总览
- 错题管理
- 复习中心
- 出卷自测
- 在线AI解题
- 个人中心

## 适用场景

- 适用于Python编程学习者的日常练习和复习
- 适合用于编程教学，辅助教师进行错题管理和教学
- 适用于自学编程，方便用户整理和回顾学习过程中的错题

## 技术特点

- 基于Flask框架开发，轻量级、易于扩展
- 采用MySQL数据库存储数据，保证数据安全
- 前端使用HTML、CSS和JavaScript实现用户界面交互
- 提供详细的开发文档，方便用户快速上手
